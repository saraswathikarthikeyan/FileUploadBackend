# FileUploadBackend
node application which provides endpoint to login,logout, file upload and file fetch

--This project was generated with [Angular CLI](https://github.com/angular/angular-cli) `ng new fileUpload` version 7.1.4.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Backend Server

This project requires backend server to run. The application uses node application end point for login/logout/fileuplod.
The endpoint is hardcoded currently inside the application.
